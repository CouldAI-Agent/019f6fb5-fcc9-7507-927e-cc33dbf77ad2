# Qarni Travel and Tours Agency

A cross-platform Flutter application for Qarni Travel and Tours Agency. This responsive application serves as a digital storefront, showcasing featured destinations and providing information about available travel services.

## Features
- **Responsive Layout**: Adapts gracefully to mobile, tablet, and desktop screens.
- **Hero Banner**: Engaging welcome section with primary call-to-action.
- **Featured Destinations**: Visual grid of popular travel locations with pricing.
- **Services Overview**: Highlighted services including flight booking, hotel reservations, and tour packages.
- **Cross-Platform**: Builds natively for iOS, Android, Web, macOS, Windows, and Linux.

## Tech Stack
- Framework: Flutter
- Language: Dart
- UI: Material Design 3

## Setup and Run
1. Ensure you have the Flutter SDK installed and configured.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your preferred connected device or emulator.
